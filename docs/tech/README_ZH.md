## 🛠️ 技术栈

- **前端**：Next.js 15.5.0 + React 19.1.0 + TypeScript + Tailwind CSS v4
- **后端**：Supabase（PostgreSQL + Auth + Storage + Edge Functions）
- **部署**：Vercel（推荐）
- **开发**：ESLint + Turbopack + PostCSS
